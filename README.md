# OpenFileLocation QGIS Plugin

This plugin adds a right-click context menu option to open the file location of any file-based layer in QGIS.

## Features
- Right-click a layer → **Open File Location**
- Works on Windows, macOS, and Linux
- Only appears for file-based layers (e.g., shapefiles, rasters)

## Installation
1. Create a folder named `OpenFileLocation`.
2. Place all plugin files inside it.
3. Zip the folder (optional).
4. In QGIS: Plugins → Manage and Install Plugins → Install from ZIP, or place directly in your profile plugins folder.
